Resumen de AJAX

AJAX (Asynchronous JavaScript And XML) es una técnica que permite actualizar partes de una página web sin recargarla completamente. Utiliza JavaScript junto con XMLHttpRequest o fetch para enviar y recibir datos del servidor de forma asíncrona.

Se Usa Principalmente Para

  • Cargar datos dinámicos sin refrescar la página.
  • Mejorar la experiencia del usuario en sitios web.
  • Comunicarse con APIs en segundo plano.

Ejemplo de Funcionamiento

<!DOCTYPE html>
<html>
<body>
<h2>Ejemplo AJAX</h2>
<button onclick="cargarDato()">Cargar mensaje</button>
<p id="resultado"></p>

<script>
function cargarDato() {
  const xhttp = new XMLHttpRequest();
  xhttp.onload = function() {
    document.getElementById("resultado").innerHTML = this.responseText;
  }
  xhttp.open("GET", "mensaje.txt", true);
  xhttp.send();
}
</script>
</body>
</html>

Cuando el usuario hace clic en el botón, AJAX solicita el archivo mensaje.txt al servidor y muestra su contenido en la página sin recargarla.

Recomendaciones de Uso de AJAX

  • Actualizar solo partes específicas de la página.
  • Manejar correctamente errores del servidor o conexión.
  • Mostrar indicadores de carga mientras se procesa la solicitud.
  • Evitar abusar de peticiones asíncronas.
  • Usar JSON como formato de datos por su simplicidad.
